Welcome to CakePHP ACL Plugin

This plugin is a port and update for a very old ACL plugin manager. It is a little creaky around the edge, but it gets the job done. Installation is fairly easy

$ cd cake_project
$ git submodule git@github.com:interlock/acl_plugin.git app/Plugins/Acl
$ cd app/Plugins/Acl
$ git checkout -b 2.0 2.0
